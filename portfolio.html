Customization Program




Example of creating a data entry screen for collecting information to generate Certificate of Analysis documents:
  • The program should be able to record data for generating Certificate of Analysis documents in a form format.
  • The program will automatically generate a document number based on the customer code, year, and running number. This number can be used for document tracking and retrieval.
  • Users should have the ability to edit and reprint documents as needed.

Note: The Guideline Limit data is designed to accommodate future requirements. Currently, users manually specify this information on the Certificate issuance screen. Therefore, this section will be left blank. If in the future, values are specified on this screen, when generating a Certificate, the system will default to the provided values (which can be edited).
